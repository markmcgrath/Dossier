# Interview Stories (STAR+R Bank)

A bank of prepared stories for behavioral interviews. Each story uses the STAR+R format: Situation, Task, Action, Result, Reflection.

The dossier skill (Mode 3: Interview Prep) maps these stories to likely interview questions. The more stories you have, the better the mapping.

Copy this file to `stories.md` and fill it in with your own experiences. Keep `stories.template.md` as a blank reference.

---

## [Story Title — e.g., "Cross-Team Data Migration"]

**Tags:** [leadership, technical-challenge, conflict-resolution, etc.]

**Situation:** [1-2 sentences. Set the scene — company, team, timeline, stakes.]

**Task:** [What were you specifically responsible for?]

**Action:** [What did you do? Be concrete — tools, decisions, tradeoffs.]

**Result:** [Quantify if possible. Revenue, time saved, reliability improvement.]

**Reflection:** [What did you learn? What would you do differently?]
