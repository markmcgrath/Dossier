# Candidate Profile: [Your Name]

This is the *preference* layer that sits above your CV. The CV says what you
*can* do. This file says what you *want* to do. The dossier skill uses
both: the CV for capability fit, this profile for desirability fit. A role
you could do well is not the same as a role you want.

Fill in every section. The more specific you are — especially about roles to
avoid — the sharper every evaluation, search, and cover letter becomes.
